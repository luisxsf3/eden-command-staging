Atualizar dados das lojas

Cole um JSON com os novos dados ou envie um arquivo CSV. Os dados são mesclados por loja + período — histórico não é apagado.

Formato JSON

Uma linha por loja por mês. Período no formato AAAA-MM.

{"gestao_lojas":[
  {"nome":"Kock & Borges",
   "bandeira":"Edificação",
   "cidade":"S. J. Pinhais","uf":"PR",
   "periodo":"2026-05",
   "faturamento":720000,
   "cmv":0.6300,
   "ticket":64.50,
   "clientes":11000}
]}

Formato CSV

Headers obrigatórios na primeira linha. Baixe o modelo para garantir a estrutura certa.

nome,bandeira,cidade,uf,periodo,faturamento,cmv,ticket,clientes
